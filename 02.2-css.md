# Summary of CSS Responsive Design


## 1. Responsive Web Design (RWD)

RWD is an approach to serve different layouts for different screen sizes, optimizing viewing experiences across mobile, desktop, tablet, and TV devices. It's accomplished using CSS Grid, Flexbox, and media queries, with mobile-first layouts working well across all screen widths.

## 2. Flexbox

Flexbox provides an efficient way to create one-dimensional layouts (either row or column).

**Basic Setup:**

```css
.flex-container {
  display: flex;
  gap: 1rem;
  justify-content: center;
}
```

**Key Container Properties:**

```css
/* Direction */
.container {
  flex-direction: row; /* default - horizontal */
  /* or */
  flex-direction: column; /* vertical */
}

/* Wrapping */
.container {
  flex-wrap: wrap; /* allows items to wrap onto multiple lines */
}

/* Main axis alignment */
.container {
  justify-content: start; /* default */
  /* or center, space-between, space-around, etc. */
}

/* Cross axis alignment */
.container {
  align-items: stretch; /* default */
  /* or center, start, end, baseline */
}

/* Multiple line alignment */
.container {
  align-content: stretch; /* default */
  /* controls spacing between wrapped lines */
}
```

**Flex Item Properties:**

```css
.item {
  order: 2; /* changes display order */
  flex-basis: 200px; /* default size before space distribution */
  flex-grow: 1; /* allows item to grow */
  flex-shrink: 1; /* allows item to shrink */
  align-self: center; /* overrides container's align-items */
}
```

## 3. CSS Grid

CSS Grid is a two-dimensional layout system for designing page layouts with columns and rows.

**Basic Grid Setup:**

```css
.page {
  display: grid;
  grid-template-columns: 2fr 1fr 1fr; /* 3 columns */
  grid-template-rows: auto 1fr 3fr; /* 3 rows */
  grid-gap: 0.5rem; /* space between grid tracks */
}
```

**Placing Items Using Grid Lines:**

```css
.head {
  grid-column: 2/4; /* start at line 2, end at line 4 */
  grid-row: 1/2;
  /* or use span: grid-column: 2 / span 2; */
  /* span all columns: grid-column: 1 / -1; */
}
```

**Placing Items Using Grid Areas:**

```css
.container {
  display: grid;
  grid-template: auto 1fr auto / auto 1fr auto;
  grid-template-areas:
    "header header header"
    "left-side main right-side"
    "footer footer footer";
}

header {
  grid-area: header;
}

.left-side {
  grid-area: left-side;
}

main {
  grid-area: main;
}
```

**Grid Alignment:**

```css
.container {
  /* Align items within their cells */
  justify-items: center; /* row axis */
  align-items: center; /* column axis */
  
  /* Align entire grid within container */
  justify-content: center; /* row axis */
  align-content: center; /* column axis */
}

.item {
  /* Individual item alignment */
  justify-self: center; /* row axis */
  align-self: center; /* column axis */
}
```

## 4. Media Queries

Media queries enable responsive layouts for different screen sizes.

```css
@media screen and (min-width: 700px) {
  .page {
    display: grid;
    grid-template-columns: 2fr 1fr;
    grid-template-areas: 
      "title title"
      "main header"
      "main sidebar"
      "footer footer";
  }
}
```

## 5. Common Layout Patterns

### Menu Using Flexbox

```css
nav ul {
  width: 90%;
  display: flex;
  column-gap: 1rem;
  row-gap: 0.4rem;
  flex-wrap: wrap;
}

nav ul li {
  list-style: none;
}
```

### Line-up Card

```css
.card {
  display: flex;
  flex-direction: column;
  justify-content: space-between; /* pushes first and last items to edges */
}
```

### Aspect Ratio Image Card

```css
.card img {
  aspect-ratio: 16 / 9; /* maintains 16:9 ratio when resizing */
}
```

### Clamping Card

```css
.card {
  width: clamp(23ch, 40%, 46ch); /* min, actual, max */
}
```

### Deconstructed Pancake

```css
.container {
  display: flex;
  flex-wrap: wrap;
}

.box {
  flex: 1 1 150px; /* grow, shrink, base-width */
}
```

### Pancake Stack (Header-Main-Footer)

```css
.container {
  display: grid;
  grid-template-rows: auto 1fr auto; /* header, main, footer */
  min-height: 100vh;
}
```

### Sidebar & Content

```css
.container {
  display: grid;
  grid-template-columns: minmax(100px, 20%) 1fr;
  /* sidebar min 100px, max 20%; main takes rest */
}
```

### Classic Layout (Header-3 Columns-Footer)

```css
.container {
  display: grid;
  grid-template: auto 1fr auto / auto 1fr auto;
  /* rows: header, content, footer / columns: left, main, right */
}
```

### RAM (Repeat, Auto, Minmax)

```css
.container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(280px, 1fr));
  /* automatically creates responsive columns */
}
```

### Super Centered

```css
.container {
  display: grid;
  place-items: center; /* centers both horizontally and vertically */
}
```

## Key Takeaways

- **Grid** is best for two-dimensional layouts dividing pages into sections
- **Flexbox** is ideal for one-dimensional layouts with space allocation
- **Media Queries** enable responsive design adaptations
- Grid and Flexbox can be combined (Grid for macro layouts, Flexbox for micro layouts)
