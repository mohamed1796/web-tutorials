# CSS Fundamentals Summary

This document covers the core concepts of Cascading Style Sheets (CSS) for styling web pages. Here's a comprehensive summary with code examples:

## 1. CSS Syntax

CSS rules consist of selectors and declarations. Each declaration contains a property-value pair.

```css
/* Basic CSS syntax */
selector {
    property: value;
    another-property: another-value;
}

/* Example */
h1 {
    color: blue;
    font-size: 24px;
}
```

## 2. Ways to Incorporate CSS

**Inline CSS:**
```html
<p style="color: red; font-size: 16px;">This is inline styling</p>
```

**Embedded CSS:**
```html
<head>
    <style>
        p {
            color: blue;
            font-size: 14px;
        }
    </style>
</head>
```

**External CSS (Recommended):**
```html
<head>
    <link rel="stylesheet" href="styles/styles.css">
</head>
```

## 3. The Cascade

Later styles override earlier ones in case of conflict. Inline styles have the highest priority.

```html
<link rel="stylesheet" href="styles/qu-styles.css">
<link rel="stylesheet" href="styles/ceng-styles.css">
<link rel="stylesheet" href="styles/cse-styles.css">
<!-- cse-styles wins in conflicts -->
```

## 4. Inheritance

Child elements inherit properties from parent elements.

```css
body {
    font-family: Arial;
    font-size: 16px;
    color: black;
}

h1 {
    color: red; /* Overrides inherited color */
}
```

## 5. CSS Selectors

**Element Selector:**
```css
p {
    color: blue;
}
```

**Class Selector:**
```css
.highlight {
    background-color: yellow;
}

/* Multiple classes on one element */
.bold {
    font-weight: bold;
}
```

```html
<p class="highlight bold">Text with multiple classes</p>
```

**ID Selector:**
```css
#header {
    background-color: navy;
}
```

**Universal Selector:**
```css
* {
    margin: 0;
    padding: 0;
}
```

## 6. Combined Selectors

**Descendant Selector:**
```css
/* Selects all <a> inside <li> */
li a {
    text-decoration: none;
}
```

**Multiple Selectors:**
```css
div, p {
    margin: 10px;
}
```

**Direct Child:**
```css
div > p {
    color: green;
}
```

## 7. Attribute Selectors

```css
/* Starts with */
a[href^="https"] {
    color: green;
}

/* Ends with */
a[href$=".pdf"] {
    font-weight: bold;
}

/* Contains */
a[href*="qu"] {
    text-decoration: underline;
}
```

## 8. Pseudo-classes

```css
/* Link states */
a:link {
    color: blue;
}

a:visited {
    color: purple;
}

a:hover {
    color: red;
}

a:focus {
    outline: 2px solid orange;
}
```

## 9. Structural Pseudo-classes

```css
/* First and last child */
tr:first-child {
    font-weight: bold;
}

tr:last-child {
    border-bottom: 2px solid black;
}

/* Nth child */
tr:nth-child(2) {
    background-color: lightgray;
}

/* Odd and even rows */
tr:nth-child(odd) {
    background-color: #f9f9f9;
}

tr:nth-child(even) {
    background-color: white;
}
```

## 10. Has Selector

```css
/* Style div that contains a p element */
div:has(p) {
    background-color: green;
}
```

## 11. Pseudo-elements

```css
/* First letter styling */
p::first-letter {
    font-size: 2em;
    font-weight: bold;
}

/* Insert content */
p::before {
    content: "«";
}

p::after {
    content: "»";
}
```

## 12. Text Properties

```css
p {
    color: #333;
    text-align: center;
    text-decoration: underline;
    text-transform: uppercase;
    line-height: 1.5;
    letter-spacing: 2px;
    word-spacing: 5px;
}
```

## 13. Font Sizing

**Fixed Sizing:**
```css
h1 {
    font-size: 24px;
}
```

**Relative Sizing:**
```css
body {
    font-size: 16px; /* Base size */
}

h1 {
    font-size: 1.5em; /* 1.5 times parent */
}

p {
    font-size: 1rem; /* Relative to root element */
}
```

## 14. List Styles

```css
ul {
    list-style-type: square;
}

ol {
    list-style-type: upper-roman;
}

/* Remove list styling */
ul.no-bullets {
    list-style-type: none;
}
```

## 15. Box Model

```css
.box {
    /* Content */
    width: 300px;
    height: 200px;
    
    /* Padding - space inside border */
    padding: 20px;
    
    /* Border */
    border: 2px solid black;
    
    /* Margin - space outside border */
    margin: 10px;
}
```

## 16. Margin and Padding Shorthand

```css
/* All four sides */
.box1 {
    margin: 10px;
}

/* Top/bottom and left/right */
.box2 {
    padding: 10px 20px;
}

/* Top, right, bottom, left (clockwise) */
.box3 {
    margin: 1px 3px 5px 7px;
}
```

## 17. Borders

```css
/* Individual properties */
.border1 {
    border-width: 1px;
    border-style: solid;
    border-color: red;
}

/* Shorthand */
.border2 {
    border: 1px solid red;
}

/* Different sides */
.border3 {
    border-top: 2px dashed blue;
    border-bottom: 1px solid gray;
}
```

## 18. Rounded Corners & Centered Box

```css
.centered-rounded-box {
    display: flex;
    column-gap: 1rem;
    align-items: center;
    padding: 0.5rem;
    border: 1px solid saddlebrown;
    border-radius: 15px;
    width: 80%;
    margin: auto; /* Centers the box */
}
```

---

**Key Takeaways:**
- CSS separates content (HTML) from presentation
- External stylesheets are preferred for maintainability
- Specificity determines which styles are applied (ID > Class > Element)
- The box model includes content, padding, border, and margin
- Relative units (em, rem) are better for responsive design
